2.0 - Helpers

Various helper classes used throughout the site

2.1 - Positioning

The dummy class is only for presentation purposes and should be ignored.

  • .center -

    Centers an element. The element needs to a have a width defined.

  • .left -

    Floats the element left.

  • .right -

    Floats the element right.

  • .flex-center -

    Center aligns items inside of it with flex

{$modifiers}
.center
center
.left
left
.right
right
.flex-center
flex-center
<div class="{$modifiers} dummy">{$modifiers}</div>

2.2 - Spacing

Spacers are used to positioning specific elements. Usually it can be modules or elements that just need a litle nudge in a specific direction and a whole module modifier class is overkill for it.

  • .flush -

    Clear the bottom margin.

  • .slim -

    Half spacer unit for the button $spacing / 2

  • .landmark -

    Usefull for hero elements. Gives element a double bototm margin.

  • .row -

    Turns element into a block element and gives it a bottom margin.

{$modifiers}
.flush
flush
.slim
slim
.landmark
landmark
.row
row
<div class="{$modifiers}">{$modifiers}</div>

2.3 - Text Helpers

Various helpers for text element

  • .align-right -

    Align text right

  • .align-center -

    Align text center

  • .align-left -

    Align text left

  • .caps -

    All CAPS text

  • .capitalize -

    Capitalize text

  • .muted -

    Muted text

  • .bold -

    Makes text bold weight

  • .normal -

    Makes text normal weight

  • .light -

    Makes text light weight

2.4 - Misc. Helpers

The dummy class is only for presentational purposes.

  • .divider -

    Divider class. Adds a border to the bottom of the element

  • .square -

    Makes the element behave a like a square image and retains its dimensions

  • .rectangle -

    Makes the element behave a like a rectangle image and retains its dimensions

  • .cursor -

    Change curser to pointer
